# Ω// LeetGame 黑核协议

一个可直接开箱即用的 ASCII / 终端风 LeetCode 克隆站小游戏。

你会先看到一个经典的 `donut.c without a math library` 甜甜圈入口页，点击后进入主游戏页面。主游戏采用黑绿青黄的故障码风，支持随机出题、分层推进、种子控制、自定义题库，并带有 Ω 世界观叙事。

## 功能介绍

本项目从使用深度上可以分成四个层级：

### 1. 入口页
- 保留原版 `donut.c` 甜甜圈风格
- 点击甜甜圈进入主游戏
- 支持 `Enter` 键进入
- 用最小交互完成“进入世界”的仪式感

### 2. 主游戏页
- 随机题流推进
- 题型包括：单选、多选、填空、BOSS 关
- 支持三层难度：`easy / medium / hard`
- 支持种子复现
- 支持导入自定义题库 JSON
- 支持导出当前题库
- 带有日志 trace、熵值、角动量等状态反馈

### 3. 叙事与风格
- Ω 语言叙事
- 终端 / CLI / agent trace 风格
- 黑客控制台视觉
- 适合做开源展示、本地小游戏演示、静态站点部署

### 4. 扩展能力
- 题库可替换
- 题目难度可扩展
- 可继续往动画、音效、剧情分支方向增强

## 安装方法

### 方法 1：直接打开
双击 `index.html` 即可开始。

### 方法 2：本地静态服务器
如果你希望跨浏览器更稳定，建议使用本地静态服务器：

```bash
python -m http.server 8000
```

然后访问：

```text
http://localhost:8000/index.html
```

## 玩法说明

### 入口阶段
- 你会先看到甜甜圈 ASCII 动画
- 点击后进入主游戏
- `Enter` 键也可以进入

### 主游戏阶段
- 每轮会随机抽取一道题
- 先完成题目，再点击 `Next Node` 推进
- 题目正确会降低 `entropy`
- 错误会降低 `momentum`
- `momentum` 归零后游戏结束

### 题库
内置题库覆盖了常见算法方向：
- 数组
- 字符串
- 栈 / 队列
- 链表
- 树 / BST
- 二分
- 滑动窗口
- 图
- 动态规划
- 区间
- 系统设计
- 调试与工程边界

### 自定义题库格式
你可以在页面中粘贴 JSON 题库，格式如下：

```json
{
  "easy": [
    {
      "type": "single",
      "category": "array",
      "question": "...",
      "options": ["A", "B", "C", "D"],
      "correct": 1,
      "explanation": "...",
      "hint": "...",
      "tags": ["array"]
    }
  ],
  "medium": [],
  "hard": []
}
```

#### 支持的题型
- `single` 单选
- `multi` 多选
- `fill` 填空
- `boss` BOSS 关卡（表现形式仍是选择题结构）

## 交互说明

### 入口页
- 点击甜甜圈进入
- `Enter` 键也可以进入

### 主游戏页
- 点击选项作答
- 填空题输入答案
- `Submit` 提交答案
- `Next Node` 进入下一题
- `Hint` 查看提示
- 左侧可以加载种子和自定义题库
